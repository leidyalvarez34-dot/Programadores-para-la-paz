Entidad: Causa

Campos principales:
- id
- titulo
- descripcion
- fecha
- estado

Propósito:
Guardar las causas comunitarias que serán visibles en la plataforma para que la comunidad pueda conocerlas, revisarlas y participar.


Entidad: Apoyo

Campos principales:
- id
- causa_id
- usuario
- fecha

Propósito:
Registrar los apoyos que recibe cada causa para hacer seguimiento a la participación ciudadana.


Entidad: Usuario

Campos principales:
- id
- nombre
- correo
- rol

Propósito:
Identificar a las personas que usan la plataforma y controlar qué acciones puede realizar cada una según su rol.
